# Nourish 🌱

A psychology-informed nutrition tracker designed for real life.

## Features

- **AI Food Recognition** — Snap a photo or describe your food by voice
- **Mood & Hunger Check-ins** — Track how food makes you feel
- **Intermittent Fasting Timer** — Flexible protocols designed for shift workers
- **AI Coach** — Personalised insights based on your patterns
- **Craving Interceptor** — Understand what's really driving the urge
- **Bad Day Mode** — No goals, no guilt — just getting through today
- **Detailed Insights** — Health signals, macros, and progress tracking

## Tech Stack

- React + Vite
- Hosted on Vercel

## Getting Started

```bash
npm install
npm run dev
```
